# mitzi-snowflake
A simple Flipper Zero app to create nice-looking 60° or 120° graphics. 
Watch the iterative growth of a snowflake on 64x64 pixels. 

<img alt="Flipper Zero screenshot"  src="screenshots/MainScreen.png" width="40%" />


## Usage
* **OK** button to grow the snowflake step by step
* **Back** button exits the game
